## 架构说明
- OpenClaw `--profile` 是进程级隔离（独立 gateway 实例），不适合单机 cron 隔离
- 所有 cron 任务已使用 `sessionTarget: "isolated"`，天然与 main session 隔离
- 每次 cron 执行都是独立 session，不继承历史上下文
